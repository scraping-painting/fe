## React Starter

### Commands

Run dev:

```bash
$ pnpm dev
```

Run build:

```bash
$ pnpm build
```

Run linting:

```bash
$ pnpm lint # show eslint errors

$ pnpm lint:fix # fix eslint errors
```

### Features

- Routing
- Layouts, pages and components
- Redux store
- Axios
- Multi languages
- Custom hooks
- Storybook
- Unit test (includes mock API test)
- TypeScript
- SCSS
